WEATHER API WITH ZIP CODE TRAVEL SUGGESTIONS AND COMPARES CITIES

Project Description:
This project is a weather API that provides weather information based on zip codes. It also offers travel suggestions and compares weather conditions between different cities. The API allows users to retrieve current weather data, forecasts, and travel recommendations based on the weather conditions in various locations.

Features:
1. Weather Data Retrieval: Users can input a zip code to get current weather information, including temperature, humidity, wind speed, and weather conditions.
2. Travel Suggestions: Based on the weather conditions in a given location, the API provides travel suggestions, such as recommended activities, places to visit, and packing tips.
3. City Comparison: Users can compare the weather conditions of two or more cities to help them decide on travel destinations or plan their trips accordingly.
4. Forecasting: The API provides weather forecasts for the next few days, allowing users to plan their activities and trips in advance.
5. User-Friendly Interface: The API is designed to be easy to use, with clear endpoints and documentation for developers to integrate it into their applications.
6. radar and satellite imagery: The API also includes radar and satellite imagery to provide users with a visual representation of weather patterns and conditions in different locations.
7. Clickable Map: The API includes a clickable map feature that allows users to select a location by clicking on the map, which then retrieves the weather information for that specific area.


Technologies Used:
- HTML
- CSS
- JavaScript
